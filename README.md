# My First GitHub Project

Simple "Hello World" webpagina, automatisch gedeployed via GitHub Actions naar GitHub Pages.

## Live
- https://egschipp.github.io/my-first-github-project/

## Hoe werkt de deploy?
Elke push naar de `main` branch triggert de workflow in `.github/workflows/pages.yml`.
Die bouwt een simpele static site en deployt naar GitHub Pages.

## Lokaal testen
Open `index.html` in je browser.
